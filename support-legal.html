How to Report a Bug

When reporting a bug, please provide as much detail as possible to help us identify and resolve the issue quickly. The more information you provide, the faster we can fix the problem.

Bug Report Form

Bug Report Guidelines

To help us resolve your issue quickly, please include:

  • Detailed Description: Explain what happened and what you expected to happen
  • Steps to Reproduce: List the exact steps that led to the bug
  • Browser Information: Include your browser name and version
  • Operating System: Specify your OS and version
  • Error Messages: Copy any error messages you received
  • Video URL: If the issue is with a specific video, include the URL
  • Screenshots: If possible, attach screenshots showing the issue

Common Issues and Solutions

Download Not Starting

  • Check if the YouTube URL is valid and accessible
  • Try refreshing the page and attempting the download again
  • Disable browser extensions that might interfere with downloads
  • Clear your browser cache and cookies

Poor Video Quality

  • Ensure the original video supports the quality you're trying to download
  • Try selecting a different quality option
  • Check if the video is still available on YouTube

Browser Compatibility Issues

  • Update your browser to the latest version
  • Try using a different browser (Chrome, Firefox, Safari, Edge)
  • Disable ad blockers temporarily
  • Enable JavaScript in your browser settings

Bug Status Tracking

  2. Investigate: Our development team will analyze the issue
  3. Update: Keep you informed about the progress
  4. Resolve: Deploy a fix and notify you when it's available
